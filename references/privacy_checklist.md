# Privacy Checklist

Before moving any internal Brainstorm Skill material into this public repository, remove:

- local absolute paths
- private workspace names
- private transcripts
- private SRT files
- personal chat logs
- internal score notes
- unreleased workflow reports
- screenshots that reveal private files
- video exports from private projects
- API keys, tokens, cookies, emails, phone numbers
- private Notion, Drive, Pencil, MCP, or automation details

Replace risky material with:

- synthetic notes
- anonymous SRT snippets
- template placeholders
- public-safe abstract examples
